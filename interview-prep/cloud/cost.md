# Cost Optimisation Q&A

> **Cloud cost optimisation and FinOps questions.**

!!! question "Q: What are the top 5 ways to reduce AWS costs?"
    1. **Right-sizing** — Match instance types to actual workload requirements
    2. **Reserved Instances / Savings Plans** — Commit for 1-3 years for up to 72% discount
    3. **Spot Instances** — Use for fault-tolerant workloads (up to 90% savings)
    4. **Auto-scaling** — Scale down during off-peak hours
    5. **Storage tiering** — Move infrequently accessed data to S3 Glacier
